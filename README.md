# Phantasia
A Discord Bot

## Requirements
 -  NodeJS 8.x or higher

## Dependencies
 - discord.io (latest)
 - winston 3.0.0

## Setup
1. Copy `config.example.json` to `config.json`
2. Edit `config.json` and change the value of `token` to your Discord bot's Token
3. Run the command `npm install` in the folder 

## Running the Bot
1. Run `node bot.js`
